Dealseeker was a really interesting group project in which each member of the group was given a specific role.

My role was focused on working closely with the Google Maps API to create all the features and interactions with the map.

The main tools and technologies I used in this project were the following:

  • Javascript
  • Jquery
  • Google Maps API v3

I also made use of the GeoJSON format for encoding a collection of markers (points) which were then loaded onto the map.

The project was hosted on Github and each member of the group created a personal branch to work with. "Git cycles" were done regularly where branches were merged with the master branch.

{% comment %} STREET JACKETS {% endcomment %} {% elsif proj_url == 'sj' %}

Street Jackets is a real life food truck business based in Wanganui.

The owners wanted a simple and nice looking website where they could show their schedule, food menu and contact details.

The website features:

  • a responsive one page layout,
  • a responsive slideshow with swipe slide enabled for touch devices,
  • a jquery slide transition effect when navigating through different sections and
  • a fixed top navigation bar (Back to Top) created on the fly (JQuery) when the user scrolls down the page.

The website was created from zero using Sublime Text Editor and Bootstrap for layout.

All the JS/JQuery functions were placed in a separate file to make it unobstrusive and a progressive enhancement strategy has also been applied.

{% comment %} MOVIE WEBSITE {% endcomment %} {% elsif proj_url == 'jp' %}

In this project I made used of HTML5 and CSS3 elements and properties such as:

  • video: for the movie trailer section,
  • transform + rotate: to create the rotate effect in the menu items,
  • transform + perspective: to keep the size of the rotating 'box' the same during the transition
  • transition: to animate the changes from the initial state to the hover state and viceversa.

I also integrated a JQuery plugin called "magnific popup" to create the modal windows used in the "Scenes" and "The Dinosaurs" sections.

In addition, I reused the scroll effect and fixed top navigation bar that I implemented for the Street Jackets project.

For the responsive grid based layout I used Bootstrap.
